## 📢 简介

OrangeQuiz（橘子测试）是一款基于 [AstrBot](https://github.com/Soulter/AstrBot) 框架的LLM驱动的问卷测试娱乐插件。依托大语言模型的聚合与逻辑泛化能力，它能够根据用户仅提供的一句灵感描述，瞬间从无到有地生成高质量的心理测试、情境抉择、人品测算或搞怪问卷。

本插件能智能调度题型（支持**计分类**或**盲盒随机抽签类**），也可为每一位完成测试的玩家渲染专属的**高质量带图“打分结果档案海报”**。结合 LLM 的强力加持，最终的结果点评还能严格对应你指定的 AI 语气（如：毒舌犀利、极致温柔、发疯文学），堪称活跃聊天氛围的最佳利器。

## ✨ 核心功能
| 功能 | 说明 |
|:---|:---|
| **AI 极速生成** | 仅需一句想法即可全自动构思专业考题与逻辑选项，内置生成过程交互退回微调闭环。 |
| **双模渲染黑科技** | 内置基于 Playwright 的高逼格浏览器快照模块，生成邀请海报与问卷结果海报。 |
| **智能吐槽语料库** | 测试结束后由 LLM 根据玩家真实的选项轨迹，生成指定语气的人格化评价。 |
| **反刷屏与安全分流** | 可灵活配置问卷游玩与新建环节的“群组防刷屏隔离开关”，只将高潮海报留给群聊，答题留给私信。 |
| **沙盒化数据库热管理** | 问卷元数据与玩家游玩轨迹由 JSON 和本地 SQLite `data` 目录无感建档热记录，署名追溯到出题作者。 |

### 1. 安装
在 AstrBot 插件管理面板或扩展商店中，搜索并安装 `astrbot_plugin_orangequiz`。

### 2. 前置条件
**基础大语言模型** —— 插件极度依赖您已关联的大模型来进行题目逻辑架构化排版。请优先保证选用的 LLM 具备标准且稳健的指令遵从及 JSON 回复能力（推荐 GPT-4o、DeepSeek 等拥有深厚逻辑调教基底的模型）。

### 3. 开始使用
问卷创建及所有的调度与游玩，一切动作均只依靠普通对话触发。
使用 `/quiz create` 让 AI 按你的旨意编写一套测试，确认发布后即可获得一条包含 123456（6位专有 ID）格式的发布海报。接着发送 `/quiz 123456`即可立即体验。

## 🛠 控制面板设置

所有的基础规则及外观自定义，均可在 AstrBot 可视化控制面板中热修改或是在配置文件里变更，修改即时生效：

### 常规测试配置
| 配置项 | 默认值 | 说明 |
|:---|:---|:---|
| **允许在群聊中互动答题** | `false` | `allow_group_quiz`。✅ 开启: 不受限制。<br>❌ 关闭: 群里输入指令会被请出转为私聊防刷屏。但已做完测试生成的海报仍可在群聊发送。 |
| **允许在群聊中出题** | `false` | `allow_group_create`。✅ 开启: 不受限制。<br>❌ 关闭: 保护群内文字空间，要求转移制卷环境。 |
| **仅限管理员新增问卷** | `false` | `admin_only_create`。✅ 开启: 只有管理员出题并进库，禁止普通水友操作避免数据爆炸。 |
| **每日生成测试次数上限** | `3` | `daily_create_limit`。每个普通用户每天能够挥霍算力生成问卷的数量。（Bot 管理员可无视此限制）。 |
| **每次问卷修改次数上限** | `8` | `max_modify_count`。在生成一份新问卷草稿后，最多允许与 AI 发生多少次退回修改及重制要求对话。 |

### 海报模板参数
| 配置项 | 默认值 | 说明 |
|:---|:---|:---|
| **海报说明文案** | `对bot发送以上带有编号的指令，开始同款测试，分享测试结果！` | `invite_poster_tip`。展示在新问卷发布宣传海报中间，以及专属定论结果海报测试胶囊按钮下方的行为引导语。<br>可用 `{test_id}` 代替问卷专属号码。 |
| **海报底部水印文字** | `可以和bot私聊参与测试...` | `footer_text`。自定义展示在一系列宣发截图最底部的引流或版权注释。 |

## 🎮 指令速查列表

所有常规交互默认使用 `/` 或你系统指定的触发前缀。同时也内置了丰富的中文常用短词方便用户快速发起测试（例如发送【答题】）。

### 日常使用（全员支持）
| 指令/正则 | 也可以输入 | 说明 |
|:---|:---|:---|
| `/quiz list` | `/测试列表`, `/问卷列表` | 检阅系统已收录并上架的所有现存问卷及其作者署名与简介。 |
| `/quiz hot` | `/热门测试`, `/测试排名` | 展示大家游玩次数最多的爆款 Top 5。 |
| `/quiz [编号]` | `/测试 [编号]` | 一键直达！进入对应题号的问卷答题状态（未指定编号时将引导您选择）。 |
| `/quiz [编号] retry`| `/测试 [编号] retry` | 不服 AI 的毒舌评价？刷新纪录再战一次该问卷。 |
| `/quiz stop` | `/退出测试`, `/结束答题`, `取消`, `退出`| 随心所欲在任何阶段立刻掐断流程进程。 |
| `/quiz help` | `/测试帮助`, `/答题帮助` | 一键输出群组机器人使用说明。 |

### 出题创作（视预留配置进行门禁检测）
| 指令/正则 | 也可以输入 | 说明 |
|:---|:---|:---|
| `/quiz create` | `/创建测试`, `/新增问卷`, `/出题` | 进入向导式出题对话室，在与小助手多轮对话后一键部署新鲜出炉的测评标杆！ |

*注意：通过指令带参支持带空格查阅如：`/quiz 123 456` 获取更易读的编码；使用指令前缀等原生条件作为调用指令依框架特性自适应，如未设置强制前缀直接喊话同样响应。*
