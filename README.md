## 📢 简介

OrangeQuiz（橘子测试）是一款基于 [AstrBot](https://github.com/Soulter/AstrBot) 框架的LLM驱动的问卷测试娱乐插件。依托大语言模型的聚合与逻辑泛化能力，它能够根据用户仅提供的一句灵感描述，瞬间从无到有地生成高质量的心理测试、情境抉择、人品测算或搞怪问卷。

本插件能智能调度题型（支持**计分类**、**盲盒随机分配类**，以及支持0题目群聊群发的**一秒纯抽卡类**），也可为每一位完成测试的玩家渲染专属的**高质量带图“打分结果档案海报”**。结合 LLM 的强力加持，最终的结果点评还能严格对应指定的 AI 语气（如：毒舌犀利、极致温柔、发疯文学）。

## ✨ 核心功能
| 功能 | 说明 |
|:---|:---|
| **AI 极速生成问卷** | 仅需一句想法即可全自动构思专业考题与逻辑选项，内置生成过程交互退回微调闭环。 |
| **群聊一键抽卡机** | 全新纯抽卡题型支持：例如今日运势/塔罗牌，无需答题，群聊输入 `/quiz [编号]` 一秒直出专属结果海报！ |
| **主题渲染** | 内置独立可配置的主题 HTML 系统模板，生成绝赞的邀请海报与问卷结果海报。 |
| **智能吐槽** | 测试结束后由 LLM 根据玩家真实的选项轨迹，生成指定语气的人格化评价。 |
| **反刷屏与安全分流** | 可灵活配置问卷游玩与新建环节的“群组防刷屏隔离开关”，只将高潮海报留给群聊，答题留给私信。 |
| **沙盒化数据库热管理** | 问卷元数据与玩家游玩轨迹由本地 SQLite 无感建档安全记录，支持每日配额计次。 |

### 1. 安装
在 AstrBot 插件管理面板或扩展商店中，搜索并安装 `astrbot_plugin_orangequiz`。

### 2. 前置条件
**基础大语言模型** —— 插件极度依赖Astrbot设置的默认语言模型来进行题目逻辑架构化排版，故请优先保证选用的 LLM 具备标准且稳健的指令遵从及 JSON 回复能力（推荐 GPT、DeepSeek 等拥有深厚逻辑调教基底的模型）。

### 3. 快速使用说明
问卷创建及所有的调度与游玩均只依靠普通对话触发。
使用 `/quiz_create` 让 AI 按你的旨意编写一套测试，确认发布后即可获得一条包含 123456（6位专有 ID）格式的发布海报。接着发送 `/quiz 123456`即可立即体验。系统自带了一套测试问卷，ID 为 000001。

## 🛠 控制面板设置

所有的基础规则及外观自定义，均可在 AstrBot 可视化控制面板中热修改或是在配置文件里变更，修改即时生效：

### 常规测试配置
| 配置项 | 默认值 | 说明 |
|:---|:---|:---|
| **允许在群聊中互动答题** | `false` | `allow_group_quiz`。✅ 开启: 不受限制。<br>❌ 关闭: 群里输入指令会被提示转为私聊防刷屏。但已做完测试生成的海报仍可在群聊发送。 |
| **允许在群聊中出题** | `false` | `allow_group_create`。✅ 开启: 不受限制。<br>❌ 关闭: 保护群内文字空间，要求转移制填写环境。 |
| **仅限管理员新增问卷** | `false` | `admin_only_create`。✅ 开启: 只有管理员出题并进库，禁止普通水友操作，避免数据爆炸。 |
| **每日生成测试次数上限** | `3` | `daily_create_limit`。每个普通用户每天能够挥霍算力生成问卷的数量。（Bot 管理员可无视此限制）。 |
| **每次问卷修改次数上限** | `8` | `max_modify_count`。在生成一份新问卷草稿后，最多允许与 AI 发生多少次退回修改及重制要求对话。 |

### 海报模板参数
| 配置项 | 默认值 | 说明 |
|:---|:---|:---|
| **海报说明文案** | `对bot发送以上带有编号...` | `invite_poster_tip`。展示在新问卷发布宣传海报中间及专属定论胶囊按钮下方的行为引导语。<br>可用 `{test_id}` 动态嵌入号码。 |
| **海报底部水印文字** | `可以和bot私聊参与测试...` | `footer_text`。自定义展示在一系列宣发截图最底部的引流或版权注释。 |
| **结果海报主题** | `default` | `result_theme`。在插件目录下的 `templates/` 系统中选用自己编写的海报前端渲染逻辑皮肤。 |
| **分享海报主题** | `default` | `invite_theme`。在插件目录下的 `templates/` 系统中选用自己编写的推荐海报皮肤。 |

## 🎮 指令速查列表
所有常规交互默认使用 `/` 或你系统指定的触发前缀。同时内置了短中文触发别名！
*Note: 指令前缀按照Astrbot有关设置确定，如设置为`!`则为`!quiz 123456`，程序的提示和生成的海报也会同步修改。*

### 日常使用（全员支持）
| 核心指令 | 别称可以直接输入（带前缀） | 说明 |
|:---|:---|:---|
| `/quiz_list` | `/测试列表`、`/问卷列表` | 检阅系统已收录并上架的所有现存问卷及其作者署名与简介。 |
| `/quiz_hot` | `/热门测试`、`/测试排名` | 展示大家游玩次数最多的爆款 Top 5。 |
| `/quiz [编号]` | `/测试 [编号]` | **核心交互**！一键直达，进入对应题号的答题状态。 |
| `/quiz [编号] retry`| `/测试 [编号] retry` | 不服 AI 的毒舌评价？刷新纪录再战一次该问卷。 |
| `/quiz_stop` | `/退出测试`、`/结束答题`、`/取消`、`/退出`| 随心所欲在任何阶段立刻终止流程事件。 |
| `/quiz_help` | `/测试帮助`、`/答题帮助` | 呼出机器人使用说明及本指南图。 |

### 出题创作（视预留门禁配置进行检测）
| 核心指令 | 别称可以直接输入（带前缀） | 说明 |
|:---|:---|:---|
| `/quiz_create` | `/创建测试`、`/新增问卷`、`/出题` | 进入向导式出题对话室，在与小助手多轮对话后一键部署新鲜出炉的测评标杆！ |

*注意：`quiz` 答题主体必须附带参数编码使用，例如：`/quiz 123456`。*
